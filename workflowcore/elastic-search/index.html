工作流核心的 ElasticSearch 插件

由 ElasticSearch 支持的工作流核心的搜索索引插件，使你能够索引你的工作流，并根据它们的数据和状态进行搜索。

正在安装

安装 NuGet 包“workflowcore.providers.elasticsearch ”

使用 NuGet 包控制台

PM> Install-Package WorkflowCore.Providers.Elasticsearch

使用.NET CLI

dotnet add package WorkflowCore.Providers.Elasticsearch

配置

构建服务提供程序时，请使用上 IServiceCollection 的 .UseElasticsearch 扩展方法

using Nest;
...
services.AddWorkflow(cfg =>
{
    ...
    cfg.UseElasticsearch(new ConnectionSettings(new Uri("http://localhost:9200")), "index_name");
});

用法

将 ISearchIndex 服务注入到代码中并使用该 Search 方法。

Search(string terms, int skip, int take, params SearchFilter[] filters)

条款

用空格分隔的搜索词字符串，空字符串将匹配所有内容。这将对以下默认字段执行全文搜索 * 参考文献 * 描述 * 地位 * 工作流定义

此外，你还可以在自己的自定义数据对象中搜索数据（如果它实现了 ISearchable

``` using WorkflowCore.Interfaces; ... public class MyData : ISearchable { public string StrValue1 { get; set; } public string StrValue2 { get; set; }

public IEnumerable<string> GetSearchTokens()
{
    return new List<string>()
    {
        StrValue1,
        StrValue2
    };    
}

} ```

##### 例子

在所有字段中搜索“小狗” searchIndex.Search("puppies", 0, 10);

Skip&Take

使用 skip 和 take 对搜索结果进行分页。其中， skip 是要开始的结果编号， take 是页面大小。

过滤器

你还可以提供应用于搜索的筛选器列表，这些筛选器既可以应用于标准字段，也可以应用于自定义数据对象中的任何字段。不需要在数据对象上实现 ISearchable，就可以对其使用筛选器。

以下过滤器类型可用 * 标量滤波器 * 数据范围过滤器 * 数值范围过滤器 * 状态筛选器

它们存在于 WorkflowCore.Models.Search 命名空间中。

##### 例子

按引用筛选 ``` using WorkflowCore.Models.Search; ...

searchIndex.Search("", 0, 10, ScalarFilter.Equals(x => x.Reference, "My Reference")); ```

按在某个日期之后启动的工作流进行筛选 searchIndex.Search("", 0, 10, DateRangeFilter.After(x => x.CreateTime, startDate));

按时间段内完成的工作流进行筛选 searchIndex.Search("", 0, 10, DateRangeFilter.Between(x => x.CompleteTime, startDate, endDate));

按处于某个状态的工作流进行筛选 searchIndex.Search("", 0, 10, StatusFilter.Equals(WorkflowStatus.Complete));

根据你自己的自定义数据类进行筛选 ```

class MyData { public string Value1 { get; set; } public int Value2 { get; set; } }

searchIndex.Search("", 0, 10, ScalarFilter.Equals(x => x.Value1, "blue moon")); searchIndex.Search("", 0, 10, NumericRangeFilter.LessThan(x => x.Value2, 5)) ```
