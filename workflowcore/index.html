工作流核心

Workflow Core 是一个面向.NET 标准的轻量级工作流引擎。思考：具有多个需要跟踪状态的任务的长时间运行的进程。它支持可插拔的持久性和并发提供程序，以支持多节点集群。

正在安装

安装 NuGet 包“WorkflowCore ”

使用 NuGet

PM> Install-Package WorkflowCore

使用.NET CLI

dotnet add package WorkflowCore

流畅的 API

使用 Fluent API 定义工作流。

public class MyWorkflow : IWorkflow
{
    public void Build(IWorkflowBuilder<MyData> builder)
    {    
        builder
           .StartWith<Task1>()
           .Then<Task2>()
           .Then<Task3>;
    }
}
